# Hospital Patient Triage & Bed Allocator

## Features

- fork() + exec()
- Shared Memory
- Named Pipes
- POSIX Threads
- Mutex & Condition Variables
- Semaphores
- Best Fit Allocation
- FCFS Scheduling
- Priority Scheduling
- Fragmentation Calculation

## Compile

make all

## Run

make run

## Stress Test

make test

## Stop

bash scripts/stop_hospital.sh
